## 说明

基于pyzmq开发的RPC服务模块，用于实现跨进程或者跨网络的服务功能调用。

## 安装

安装需要基于2.9.0版本以上的[VN Studio](https://www.vnpy.com)。

直接使用pip命令：

```
pip install vnpy_rpcservice
```

下载解压后在cmd中运行

```
python setup.py install
```
